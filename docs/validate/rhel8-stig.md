# Red Hat Enterprise Linux 8 STIG

Security Technical Implementation Guide for RHEL 8

InSpec validation profile for Red Hat Enterprise Linux 8 based on the DISA STIG requirements.

## Details

- **Maintainer:** MITRE SAF
- **Technology:** Linux
- **Category:** STIG
- **Version:** 1.12.0

## Installation

```bash
# Clone the profile
git clone https://github.com/mitre/redhat-enterprise-linux-8-stig-baseline

# Run with InSpec
inspec exec redhat-enterprise-linux-8-stig-baseline
```
