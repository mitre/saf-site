# Windows Server 2019 STIG

Security Technical Implementation Guide for Windows Server 2019

## Overview

InSpec validation profile for Windows Server 2019 based on DISA STIG requirements.

## Details

- **Maintainer:** MITRE SAF
- **Technology:** Windows
- **Category:** STIG
- **Version:** 2.8.0

## Installation

```bash
# Clone the profile
git clone https://github.com/mitre/microsoft-windows-server-2019-stig-baseline

# Run with InSpec
inspec exec microsoft-windows-server-2019-stig-baseline
```
